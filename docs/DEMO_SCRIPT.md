# Demo Script

## Flow

1. Open `/`
2. Explain: catalog of real professional website templates.
3. Open `/templates`
4. Show the two real templates.
5. Open `psicologo-clinico-agenda`
6. Click “Pedir este template”
7. Complete the pedido form
8. Submit
9. Show `/pedido/gracias?order=...`
10. Open `/admin/orders`
11. Show the order persisted with status `pending_payment`

## Message

This is not yet a fully automated website factory. It is the first commercial intake layer:

- real catalog
- real templates
- structured brief
- persisted order
- controlled operations path

Payment and automated production are intentionally separate next milestones.
